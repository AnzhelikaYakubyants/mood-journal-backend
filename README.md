# Mood Journal Backend

This repository contains the backend server with Express and Prisma ORM. 
## Getting Started

### Installation
`npm install`

### Running the server
`npm run dev` 

## `.env` example
```
DATABASE_URL="prisma+postgres://accelerate.prisma-data.net/?api_key=PLACEHOLDER_API_KEY"

JWT_SECRET="PLACEHOLDER_JWT_SECRET"
```
